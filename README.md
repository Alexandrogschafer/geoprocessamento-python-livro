# Geoprocessamento com Python

Repositório oficial do livro *Geoprocessamento com Python*.

## Estrutura de pastas

- `dados/` — shapefiles, rasters e tabelas utilizadas nos capítulos.
- `notebooks/` — notebooks do livro (um por capítulo).
- `src/` — scripts auxiliares em Python.
- `resultados/` — mapas, tabelas e figuras geradas.
- `material_adicional/` — tutoriais complementares (ex.: passo a passo do Google Colab).
